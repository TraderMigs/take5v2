# Take5 Pickleball Break

A free 5-minute pickleball mini-game and court finder for Take5.cc.

## What it includes

- Static HTML/CSS/JavaScript only
- Dark neon pickleball court design
- Mouse controls
- Keyboard controls
- Hold-to-charge power mechanic
- Score, streak, and best streak tracking
- Google Maps court search
- SEO metadata
- Sitemap and robots file
- Neuroni.dev footer CTA
- Future desktop ad slots

## Deploy target

Vercel static site connected to this GitHub repository.

## Custom domain

Primary domain: https://take5.cc/
